Node Package Manager

The NPM module creates CycloneDX BOMs from existing Node.js projects that use NPM. The module will automatically resolve all metadata for all direct and transitive dependencies.

Installing
{% highlight plaintext %} npm install -g @cyclonedx/bom {% endhighlight %}
Creating a BOM
{% highlight plaintext %} cyclonedx-bom -o bom.xml {% endhighlight %}
