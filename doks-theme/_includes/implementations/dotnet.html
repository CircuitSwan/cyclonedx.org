.NET

The .NET Core module creates CycloneDX BOMs from existing Visual Studio solutions and projects. The module will automatically resolve all metadata (via NuGet) for all direct and transitive dependencies.

Installing
{% highlight plaintext %} dotnet tool install --global CycloneDX {% endhighlight %}
Creating a BOM
{% highlight plaintext %} cyclonedx YourSolution.sln -o /output/path {% endhighlight %}
